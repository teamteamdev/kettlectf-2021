Problem

Vasya is in school and bad teached gave him a lot of homework. Vasya is lazy and he asked you for help. All tasks in his homework is quadratic equations (ax^2 + bx + c == 0) in integer numbers (Vasya don't know fractions now).

Input-Output

Vasya push to stack a b c in given order (c on top of stack). And Vasya will read results from stack. For example, if roots is 1 and 2 you should push it on stack in any order. For infinite solution push any three values.

Example

Stack before

1
-3
2 (top)

Stack after

1
2 (top)
Vasya have only primitive computer, so he can run only one rare language. Probably, you dont know it, but Vasya gave instruction.
